> AI-Powered Agricultural Assistant

Prakriti is a comprehensive platform empowering Indian farmers with:
- **AI Crop Doctor**: Diagnose diseases from photos.
- **Live Voice Chat**: Talk to Prakriti in Hindi/English/Hinglish.
- **Community**: Connect with nearby farmers.
- **Marketplace**: Buy/Sell crops and tools (Coming Soon).

## Tech Stack
- **Frontend**: React, Vite, TS, TailwindCSS
- **Backend**: Python (FastAPI), Supabase (PostgreSQL + Vector)
- **AI**: Gemini 1.5 Flash (Multimodal)

## Setup
1. `npm install` in `frontend`.
2. `pip install -r requirements.txt` in `backend`.
3. Set up `.env` with `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`, and `GEMINI_API_KEY`.
